Prosolution IA — Case Study

Resumo: SaaS para automação de postagens no Instagram com IA aplicada e monetização por planos.

O que eu construí
  • API REST com FastAPI (arquitetura modular e versionamento)
  • Integração Instagram Graph API (create → status → publish)
  • IA multi-provider (Gemini + OpenAI) com fallback e logs
  • Mercado Pago com webhooks e upgrade automático de plano
  • JWT + autorização baseada em plano
  • Persistência e auditoria (PostgreSQL + SQLAlchemy Async)
Por que não há repositório

Por confidencialidade e segurança, o código e integrações em produção não são públicos. Posso apresentar detalhes técnicos em entrevista, incluindo decisões de arquitetura e desafios resolvidos.

Prosolution IA — Arquitetura (alto nível)

  1. Client/UI → solicita ações (agendar post, gerar conteúdo, dashboard)
  2. FastAPI → auth, instagram, ai, payments (domínios separados)
  3. DB → users, instagram_posts, ai_logs, payments, reports
  4. Instagram Graph → create media → status → publish
  5. IA Providers → Gemini/OpenAI com fallback + score
  6. Mercado Pago → webhooks → upgrade plano → billing

Innovation RH — Case Study

Resumo: plataforma conceitual com foco em integração de RH, contabilidade e fiscal.

O que o sistema propõe
  • Integrações para dados de colaboradores, férias, benefícios e folha
  • Conexões com contabilidade/fiscal: DRE, balanço, apuração, obrigações
  • Coleta de dados via formulários e processos de onboarding
  • Notificações e comunicação interna
Contexto de sigilo

Estrutura exibida em alto nível. Sem dados reais e sem código público. O objetivo aqui é demonstrar domínio de arquitetura e integração.

Innovation RH — Arquitetura (alto nível)

  1. Core API → domínios: colaboradores, DP, fiscal, relatórios
  2. Integrações → RH/DP, contabilidade, fiscal, comunicação
  3. Camada de dados → normalização e validação de entradas
  4. Governança → trilhas de auditoria e rastreabilidade
